# Simple I/O Agent：单次输入输出

## 简单介绍

Simple I/O Agent 是最基础的 Agent 形式——接收用户输入，LLM 生成一次回复，任务即完成。它没有状态管理，没有工具调用，没有多步推理。本质上是一个**带系统 Prompt 的 LLM API 封装**。

## 基本原理

```
用户输入 → System Prompt + 用户消息 → 单次 LLM 调用 → 回复输出
```

## 适用场景
- 简单的问答任务
- 内容生成（翻译/摘要/写作）
- 分类和提取

## 局限
- 无法执行多步任务
- 无长期记忆
- 无法调用工具

## 工程要点
- **何时选择**：任务可以一步完成，不需要外部信息或工具
- **何时升级**：需要调用工具、需要多轮对话、需要联网搜索时
